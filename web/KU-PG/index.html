关于硕士衔接课程

硕士学术课程

对于国际学生而言，从本国的高中来到美国的大学，非常具有挑战性—新的语言环境、新的学习方式、新的生活环境。

国际学生硕士课程（MAP）将学分课程和额外的学术支持、指引和文化体验相结合，从而缓和了这一过渡。课程提供了一个以学习为主的环境，并提倡学术成就、积极参与校园生活和社区活动。

MAP 包含三种各不相同的课程：

Integrated Master’s Accelerator Program 硕士学术课程（IMAP）

Master’s Accelerator Program1 硕士学术课程一学期（MAP1）

Master’s Accelerator Program2 硕士衔接课程两学期（MAP2）
